# Login & Registration Form - React TypeScript - Redux Version

## Description

It is a React TypeScript application that allows a user to register, login, edit, and view user's pages.

## Used

- HTML

- CSS

- Bootstrap-CDN

- Redux

- Controlled Components

- Routes

- Forms

- React Hooks: useState

- Lazy Loading
